ebuild #

ebuild 是一个专注于易语言自动化构建的工具。

Synopsis #

使用Golang编写的、以 e2txt 和 ecl 作为基础工具的上层构建工具。

Options #

  -h, --help             help for ebuild
  -b, --project string   指定工程的目录。默认为当前路径。 (default "E:\\Workspace\\ebuild")
